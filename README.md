# BrilloPro 🧹✨
### Bilingual Cleaning Business Management App
**by Avanza One LLC**

> La APP para manejar tu negocio de limpieza | The app to manage your cleaning business

---

## Features
- 🗺️ **Route Optimization** — Load today's stops, optimize by ZIP code, launch GPS
- 👥 **Client Management** — Recurring schedules, lockbox codes, access notes, pet info
- 📄 **Invoices** — Create, send via text/email, mark paid, print to PDF
- 📋 **Estimates** — Build estimates, convert to invoice in one tap
- 💰 **Billing & Reports** — Export to CSV for QuickBooks/Wave, payment methods
- 🌐 **Fully Bilingual** — English / Spanish toggle throughout

## Setup
```bash
npm install
npm run dev
```

## Deploy
Connected to Vercel — push to main branch to auto-deploy.

## Image Assets
Place these files in the `/public` folder:
- `logo.png` — Square logo (app icon)
- `header.png` — Wide header banner

## Tech Stack
- React 18
- Vite
- localStorage for data persistence
- No backend required
